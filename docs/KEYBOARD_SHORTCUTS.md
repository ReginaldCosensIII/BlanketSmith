# Keyboard Shortcuts

BlanketSmith supports the following keyboard shortcuts to help speed up your workflow.

| Category | Action | Shortcut(s) |
| :--- | :--- | :--- |
| **Tools** | Brush | `B` |
| | Fill | `F` |
| | Replace | `R` |
| | Eyedropper | `I` |
| | Select | `S` |
| | Text | `T` |
| | Fill Row | `H` |
| | Fill Column | `V` |
| **Clipboard** | Copy | `Ctrl`+`C` (or `Cmd`+`C`) |
| | Cut | `Ctrl`+`X` (or `Cmd`+`X`) |
| | Paste | `Ctrl`+`V` (or `Cmd`+`V`) |
| **System** | Save Project | `Ctrl`+`S` (or `Cmd`+`S`) |
| | Undo | `Ctrl`+`Z` (or `Cmd`+`Z`) |
| | Redo | `Ctrl`+`Y` (or `Cmd`+`Y`) or `Ctrl`+`Shift`+`Z` |
| | Delete Selection | `Delete`, `Backspace` |
| | Select All | `Ctrl`+`A` (or `Cmd`+`A`) |
| **Navigation** | Zoom In | `=` , `+` |
| | Zoom Out | `-` , `_` |
| | Reset Zoom | `0` |
| **UI** | Toggle Sidebar | `Tab` |
| | Toggle UI Lock | `Ctrl`+`.` (or `Cmd`+`.`) |

> Note: Shortcuts work when the main canvas or tool area is focused. They are disabled when typing in text fields.
