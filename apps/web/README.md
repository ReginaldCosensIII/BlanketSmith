# BlanketSmith Community Platform

The future community hub for BlanketSmith.

## Purpose
This workspace (`@blanketsmith/web`) is reserved for the Next.js-based community platform features.

## Tech Stack
- **Framework**: Next.js
- **Styling**: Tailwind CSS

## Local Development

To run this workspace locally:

```bash
npm run dev:web
```

This will start the Next.js dev server.
